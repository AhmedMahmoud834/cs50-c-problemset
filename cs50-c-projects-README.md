# CS50 C Projects

A collection of problem sets completed in C as part of Harvard's CS50: Introduction to Computer Science. These projects cover fundamental programming concepts including loops, arrays, strings, and algorithm design.

## Projects

### Mario
Recreates a half-pyramid pattern using nested loops, demonstrating control flow and 2D pattern generation.

**Run:**
```bash
make mario
./mario
```

### Cash
Calculates the minimum number of coins needed to give as change, using a greedy algorithm.

**Run:**
```bash
make cash
./cash
```

### Readability
Analyzes a block of text and estimates its reading grade level using the Coleman-Liau index, based on letter, word, and sentence counts.

**Run:**
```bash
make readability
./readability
```

### Caesar
Implements a classic Caesar cipher, encrypting a message by shifting each letter by a given key.

**Run:**
```bash
make caesar
./caesar <key>
```

### Credit
Validates credit card numbers using Luhn's algorithm and identifies the card type (AMEX, MASTERCARD, or VISA).

**Run:**
```bash
make credit
./credit
```

## Requirements
- GCC compiler
- CS50 library (`cs50.h`) — install via `sudo apt install libcs50` or use the CS50 Codespace environment

## About
These projects were built while completing CS50, focusing on understanding core programming logic in a low-level, memory-managed language before moving on to higher-level languages and frameworks.
